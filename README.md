# ArtScience Entanglement Lab

### How can we archive quantum happenings?

Quantum Art Workshop · CMU Media Arts (MAD) · July 2026

เวิร์กชอปศิลปะข้ามศาสตร์ที่ใช้ **ควอนตัม** เป็นเลนส์ในการมองโลก สร้างสรรค์ และมองเทคโนโลยี
ร่วมกับทีมคิวเรเตอร์ ศิลปิน และนักฟิสิกส์ โจทย์ไม่ใช่การ "แปลง" ฟิสิกส์เป็นงานศิลป์ หรือผลิตชิ้นงาน
สำเร็จรูป แต่เป็นการเปิดพื้นที่ **co-create** ระหว่างวิทยาศาสตร์ เทคโนโลยี และศิลปะ

- **Art-Science** — จุดร่วม/จุดต่างของศิลปะกับวิทยาศาสตร์ (อาร์ตถามว่า "รู้สึกอย่างไร" วิทย์ถามว่า "จริงแค่ไหน")
